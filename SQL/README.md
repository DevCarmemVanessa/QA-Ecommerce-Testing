# SQL

Esta seção contém consultas SQL utilizadas durante a validação dos dados do sistema ShopEasy.

As consultas auxiliam na verificação da persistência dos dados, relacionamento entre tabelas e integridade das informações armazenadas.

## Artefatos disponíveis

- Consultas_QA.sql

## Objetivo

Validar informações diretamente no banco de dados, complementando os testes realizados pela interface da aplicação.

## Conceitos utilizados

- SELECT
- WHERE
- COUNT
- ORDER BY
- LIKE
- AND
- OR
- JOIN

## Exemplos de validação

Verificar usuários cadastrados.

Contar registros.

Buscar pedidos aprovados.

Validar associação entre clientes e pedidos.

Ordenar informações.

Pesquisar registros específicos.
