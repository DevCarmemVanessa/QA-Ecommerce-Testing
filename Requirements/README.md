# Requisitos do Sistema

Esta seção contém os requisitos funcionais definidos para o sistema fictício ShopEasy.

Os requisitos servem como base para elaboração dos cenários de teste, casos de teste e validação das funcionalidades.

## Artefatos disponíveis

- Requisitos_Sistema.md

## Objetivo

Definir o comportamento esperado do sistema, permitindo identificar o que deve ser desenvolvido e posteriormente validado pela equipe de testes.

## Funcionalidades documentadas

- Cadastro de usuários
- Login
- Recuperação de senha
- Catálogo de produtos
- Carrinho de compras
- Cupons de desconto
- Checkout
- Pagamentos
- Estoque
