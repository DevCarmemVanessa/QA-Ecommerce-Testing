# Execução de Testes

Esta seção apresenta os resultados obtidos durante a execução dos casos de teste do sistema ShopEasy.

A execução dos testes permite identificar funcionalidades aprovadas, falhas encontradas e possíveis defeitos a serem registrados como bugs.

## Artefatos disponíveis

- Test_Execution.md

## Objetivo

Documentar o status dos testes executados, permitindo acompanhar o progresso da validação do sistema.

## Informações documentadas

Cada execução apresenta:

- Identificador do caso de teste
- Funcionalidade validada
- Resultado obtido
- Status (Passou/Falhou)
- Relação com bugs identificados

## Possíveis Status

- Passou ✅
- Falhou ❌
- Bloqueado ⛔
- Não Executado ⏳
