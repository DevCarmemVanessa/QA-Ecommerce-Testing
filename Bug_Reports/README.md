# Bug Reports

Esta seção contém os defeitos identificados durante a execução dos testes do sistema ShopEasy.

Os bugs foram documentados utilizando boas práticas de Quality Assurance, incluindo severidade, prioridade, passos para reprodução, resultado esperado e resultado obtido.

## Artefatos disponíveis

- Bug_Reports.md

## Objetivo

Registrar defeitos encontrados durante a validação do sistema, permitindo acompanhamento, correção e reteste.

## Informações documentadas

Cada bug contém:

- Identificador único
- Título
- Severidade
- Prioridade
- Pré-condições
- Passos para reproduzir
- Resultado esperado
- Resultado obtido
