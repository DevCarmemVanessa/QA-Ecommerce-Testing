# Plano de Testes

Esta seção contém o planejamento de testes do sistema fictício ShopEasy.

O Plano de Testes define a estratégia utilizada para validar as funcionalidades do sistema, incluindo escopo, tipos de testes aplicados, riscos, critérios de entrada, critérios de saída e ferramentas utilizadas.

## Artefatos disponíveis

- Plano_de_Testes.md

## Objetivo

Organizar o processo de testes antes da execução, garantindo que as funcionalidades sejam avaliadas de forma estruturada e alinhada aos requisitos definidos.
