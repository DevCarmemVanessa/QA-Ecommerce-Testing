# Cenários de Teste

Esta seção contém os cenários de teste elaborados para o sistema ShopEasy.

Os cenários representam situações que precisam ser validadas durante os testes, permitindo verificar se as funcionalidades atendem aos requisitos definidos.

## Artefatos disponíveis

- Cenarios_de_Teste.md

## Objetivo

Identificar os fluxos principais e alternativos do sistema que devem ser avaliados durante a execução dos testes.

## Funcionalidades contempladas

- Cadastro de usuários
- Login
- Recuperação de senha
- Carrinho de compras
- Aplicação de cupons
- Checkout
- Pagamentos
