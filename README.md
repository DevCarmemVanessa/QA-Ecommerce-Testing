# QA E-commerce Testing Project 🐞

Projeto desenvolvido para demonstrar conhecimentos em Qualidade de Software (QA), abrangendo planejamento, modelagem de testes, documentação de defeitos e validação de funcionalidades em um sistema fictício de e-commerce.

## 🎯 Objetivo

Aplicar conceitos de QA Manual por meio da criação de artefatos utilizados no dia a dia de equipes de qualidade, simulando o processo completo de testes em um sistema de e-commerce.

## 🛒 Escopo do Sistema

O ShopEasy é um sistema fictício de e-commerce composto pelas seguintes funcionalidades:

- Cadastro de usuários
- Login
- Recuperação de senha
- Catálogo de produtos
- Carrinho de compras
- Aplicação de cupons
- Checkout
- Pagamentos
- Controle de estoque

## 📂 Estrutura do Projeto

### Requirements
Documentação dos requisitos funcionais do sistema.

### Test Plan
Planejamento e estratégia de testes.

### Test Scenarios
Cenários de teste derivados dos requisitos.

### Test Cases
Casos de teste detalhados.

### Test Execution
Resultados da execução dos testes.

### Bug Reports
Registro dos defeitos encontrados.

### SQL
Consultas SQL utilizadas para validação de dados.
