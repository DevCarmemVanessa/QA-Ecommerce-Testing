# QA E-commerce Testing Project 🐞

Projeto desenvolvido para demonstrar conhecimentos em Qualidade de Software (QA), abrangendo planejamento, modelagem de testes, documentação de defeitos e validação de funcionalidades em um sistema fictício de e-commerce.

## Objetivo

Aplicar conceitos de QA Manual através da elaboração de artefatos utilizados no dia a dia de equipes de qualidade.

## Escopo do Sistema

O sistema simula um e-commerce contendo funcionalidades como:

- Cadastro de usuários
- Login
- Recuperação de senha
- Catálogo de produtos
- Carrinho de compras
- Aplicação de cupons
- Checkout
- Pagamentos
- Controle de estoque

## Estrutura do Projeto

### Test Plan
Documentação do planejamento dos testes.

### Test Scenarios
Cenários de teste elaborados a partir dos requisitos.

### Test Cases
Casos de teste detalhados.

### Bug Reports
Registro de defeitos encontrados.

### SQL
Consultas SQL utilizadas para validação dos dados.

### Test Execution
Resultados da execução dos testes.
